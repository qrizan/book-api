# book-api
> Book api using Lumen

```sh
crete database = book-api
setting .env

DB_DATABASE=book-api
DB_USERNAME=root
DB_PASSWORD=

```

```sh
$ php artisan migrate:refresh --seed

$ cd public
$ sudo php -S localhost:8000

http://localhost:8000/

```
